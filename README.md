# Gerador de senhas 🔒
Um gerador de senhas simples feito com HTML, CSS e Java Script

## Funcionalidades
- Gera senhas aleatórias
- Escolha de tamanho ou quantidade de caracteres
- Opções de letras maiúsculas e minúsculas
- Números e símbolos
- Indicador de força da senha ( senha fraca, média, forte )
- Copiar senha
- Tema claro ou escuro

## Tecnologias

- HTML5
- CSS
- JavaScript
